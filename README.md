# deployment-scanner, DK WHAT TO NAME IT TBH 

AI-powered infrastructure security scanner that analyzes Kubernetes, Terraform, 
and Docker configurations in 10 seconds.

**Tech Stack:** Python, Flask, OpenRouter LLM, Render
**Features:** 40+ security rules, AI-powered remediation, CI/CD ready
**Impact:** Detects critical misconfigurations before production deployment

## kindly Note

This demo runs on Render's free tier. If inactive for 15 minutes, 
the first request may take 30-60 seconds to wake up. Subsequent requests are fast (<2s).
